# WORM GEAR RUNOUT TESTING STATION
This project demonstrates the PLC and HMI software developed for an automated Worm Gear Runout Testing Station used in manufacturing quality inspection.
The system automatically feeds parts, performs runout measurement, controls servo positioning, records test results, and provides a user-friendly HMI interface for machine operation, parameter management, and production monitoring.
The objective of this project was to improve inspection accuracy, reduce manual intervention, and provide reliable traceability of test data.

## Project Features
* Automated loading of worm gears into the inspection station
* Sensor-based part detection
* Sequential machine operation
* Servo-controlled positioning system
* Automatic test cycle execution
* Consistent and repeatable inspection process
* Test result storage
* Production tracking
* Inspection history records
* Sensor monitoring
* Alarm generation
* Safe machine recovery procedures
